## Tutorial Philosophy

These tutorials follow the Diataxis framework for learning-oriented documentation:

| Principle | How We Apply It |
|-----------|-----------------|
| **Learning by doing** | Every step has you perform an action |
| **Guaranteed success** | Exact commands and expected outputs provided |
| **Concrete steps** | No vague instructions like "configure as needed" |
| **Building confidence** | Start simple, increase complexity gradually |

Each tutorial provides:
- Clear success criteria
- Verification checkpoints to confirm progress
- Troubleshooting sections for common issues
- Screenshot placeholders for visual guidance

---

## How to Use These Tutorials

### Start from the Beginning

If you're new to the system, follow the tutorials in order:

1. **Getting Started** - Essential foundation
2. **Understanding Mastery** - Core concepts for effective learning
3. **Building Your First Deck** - Practical deck creation and adaptive experience

### Jump to a Specific Topic

If you've already completed the basics, navigate directly to the tutorial that addresses your need.

### Verification Checkpoints

Each tutorial includes verification checkpoints marked like this:

> **Verification checkpoint:** You should see [specific outcome]

Stop at these points to confirm you're on track before proceeding.

## What You Will Build

By completing all tutorials, you will have:

- A working installation of the Adaptive Mastery Learning System
- At least 5 concepts organized into a coherent study deck
- Card variants across multiple cognitive dimensions
- Experience with the complete review flow
- Understanding of the six cognitive dimensions
- Ability to interpret and act on your mastery profile
- First-hand experience with adaptive card selection
