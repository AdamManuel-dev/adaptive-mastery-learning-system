## Quick Reference

### Keyboard Shortcuts (Review)

| Key | Action |
|-----|--------|
| `Space` | Show answer |
| `1` | Again |
| `2` | Hard |
| `3` | Good |
| `4` | Easy |

### Six Cognitive Dimensions

| Dimension | Tests |
|-----------|-------|
| Definition | Term recall |
| Paraphrase | Recognition of meaning |
| Example | Instance classification |
| Scenario | Real-world application |
| Discrimination | Distinguishing similar concepts |
| Cloze | Fill-in-the-blank completion |

### Rating Guidelines

| Rating | Use when |
|--------|----------|
| Again | Completely forgot |
| Hard | Remembered with significant difficulty |
| Good | Remembered correctly with effort |
| Easy | Recalled instantly, too easy |
